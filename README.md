# cmake_cs template

cmake projects for C#

* myClassLib
  * Class library using a class in myClassLibBase as base class
* myClassLibBase
  * Class library 
* myConsoleApp
  * deps. myClassLib -> myClassLibBase
* myWinFormsApp
  * Windows forms application
* myWPFApp
  * WPF application

## limitaions

* used Visual Studio as generator
* not supported linux and dotnet
